Aplikasi Pelaporan dan Pengelolaan Data Limbah B3 Medis digunakan untuk kebutuhan operasional perusahaan dalam mengelola Data Limbah B3 Medis.
Terdapat 4 pengguna yang memiliki hak akses dan tanggung jawab yang berbeda, yaitu: 
- Administrator Sistem: melakukan pengelolaan data pengguna.
- Staf : melakukan pengelolaan data fasyankes, data limbah masuk, dan generate laporan.
- Petugas : melakukan validasi status data limbah.
- Manager : melakukan rekapitulasi limbah dan generate laporan.
